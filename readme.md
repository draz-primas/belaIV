# BelaIV

ovo je kod za robota koji igra belu  
3d printani dijelovi: https://github.com/onaj-koji-jesam/bela-IV  

## kompajliranje

```bash
git clone --recursive https://github.com/draz-primas/belaIV.git
cd belaIV
./build.sh
```

za pokretanje pokrenite `pokreni.sh`  
za automatsko pokretanje mozda trebate promjenit zadnju liniju u `pokreni.sh`
da bude cijeli path, a ne `./pokreni.sh`
